# Dini – Carpooling Web Application

Dini is a web-based carpooling platform developed as part of an academic group project (Front-End & UI/UX).  
The objective was to design, build, and deploy a modern web application using industry-standard tools and collaborative workflows.

---

## Project Context

This project was completed as a **group assignment** in an academic setting.  
It was designed to simulate a real-world development environment, with shared responsibilities across UI/UX design, frontend development, version control, and deployment.

---

## Technologies Used

- **React** – Frontend framework
- **JavaScript (ES6+)**
- **HTML5 & CSS3**
- **Figma** – UI/UX design and prototyping
- **Vercel** – Deployment and hosting
- **Git & GitHub** – Version control and collaboration

---

## Design & UX

- The **UI design** was primarily created using **Figma**.

---

## Deployment & Tooling
- The application is deployed using **Vercel**, with continuous deployment enabled through GitHub.
